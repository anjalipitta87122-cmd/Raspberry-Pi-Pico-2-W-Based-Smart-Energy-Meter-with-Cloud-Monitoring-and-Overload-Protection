# Raspberry-Pi-Pico-2-W-Based-Smart-Energy-Meter-with-Cloud-Monitoring-and-Overload-Protection
A smart energy meter with cloud monitoring and overload protection based on Raspberry Pi Pico 2 W is an Internet of Things (IoT)-enabled energy management system that allows real-time monitoring, analysis, and protection of electrical loads. This work presents the IoT-based energy management system that involves embedded systems, wireless communication, cloud computing, and web technologies.

At the heart of the energy management system is a Raspberry Pi Pico 2 W (RP2350) microcontroller used as a CPU of the board. Two PZEM-004T energy monitoring modules measure critical electrical parameters including voltage, current, power, energy consumption, frequency, and power factor for two separate AC loads. Processed data is sent wirelessly via the built-in Wi-Fi capabilities of the Pico microcontroller.

For remote control, the energy management system is also connected with the Blynk Cloud providing an opportunity for anyone to check electrical parameters of the board via their smartphones or computers. Moreover, a special web dashboard enables real-time energy consumption tracking and monitoring of electrical loads and system performance.

One of the most important aspects of the project is that of the overloading protection and faults management of the system. In case if any of the loads which is connected consumes electricity above a certain pre-defined threshold, then the load is automatically disconnected using relay control. This ensures that there will be no harm caused to the system.

This project illustrates a practical application of energy management using the Internet of Things technology in a single project where real-time sensing is used along with cloud connectivity and remote monitoring. Thus, it gives a useful insight on modern energy management systems using IoT.
